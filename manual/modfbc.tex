The {\em fbc} module provides non-adaptive feedback cancellation. As a non-adaptive method it provides two phases: during the measurement phase a test signal is played back through the loudspeaker output ports \attr{loudspeakerports}, and the response is recorded via the microphone input ports \attr{micports}. In the filter phase, the output signals are filtered with the phase-inverted corresponding responses, and the signal is added to the microphone signal.
