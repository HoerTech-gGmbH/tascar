Start system processes, e.g., to load helper programs, external decoders or video render tools.

\begin{tscattributes}
\indattr{command}   & Command to execute; process will be terminated with SIGTERM when session modules are unloaded. \\
\indattr{sleep}     & Wait for given number of seconds before proceeding with session loading.                       \\
\indattr{onunload}  & Execute this command on unload of session.                                                     \\
\indattr{id}        & OSC path prefix (default: ``system'').                                                         \\
\indattr{triggered} & Command to be executed upon OSC trigger.                                                       \\
\indattr{noshell}   & If true then do not start an additional shell to launch the command.                           \\
\indattr{relaunch}  & If true then do relaunch command when ended before session unload.                             \\
\end{tscattributes}

The commands are started into the background using this shell command line:
\begin{verbatim}
sh -c "cd sessionpath;command >/dev/null & echo \$!"
\end{verbatim}
